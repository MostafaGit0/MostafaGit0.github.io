Card Manager - Secure Card Storage App

This porject idea was actually thanks to a friend. We were trying to buy something online that didnt take apple pay (very weird) and we didnt have our wallets on us. Then he said he wishes there was a feature on our phones that saves our card details so we dont have to carry a bunch of credit cards. Then 2 days later i showed him this! Personally I've already used this countless times, espeecially when some online stores have your card info but need you to confirm the CVV. In retrospect, this would be a bit cooler as an app but I had no expiernce with that when I made it. Maybe somewhere down the line i'll make this iOS ready.

Authentication & Security:

  • Secure user authentication with Firebase Auth using username/password system
  • Users sign up with custom usernames that are converted to email format internally
  • Each user's cards are completely isolated and secured with user ID verification
  • Smooth animated login/signup interface with tab switching
  • Real-time authentication state management

UI & Visual Features:

  • Dynamic Card Type Recognition: Automatically detects card type (Visa, MasterCard, AmEx, Discover) and applies corresponding colors and animations
  • Color-Coded Cards: Visa cards display in blue gradients, MasterCard in orange, AmEx in blue-cyan, Discover in orange-red, and unknown cards match the website's theme colors
  • Pulsing Animations: Each card circle features continuous color-changing animations with smooth scaling effects that create a living, breathing interface
  • Card-Specific Icons: Each card type displays its own unique SVG icon within a floating container that animates on hover
  • 3D Hover Effects: Cards lift, rotate, and scale with advanced CSS transforms including perspective and shadow effects
  • Animated Title: The main title uses React components to animate each letter individually with rotation and blur effects
  • Ripple Effects: Hover interactions create expanding ripple animations across card surfaces

Core Features:

  • Add, view, edit, and delete credit cards with full CRUD operations
  • Real-time card number formatting with automatic spacing during input
  • Automatic expiration date formatting (MM/YY) with input validation
  • One-click copy functionality for card numbers, expiration dates, and CVV codes
  • Smart card type detection based on card number patterns
  • Secure card details view with copy-to-clipboard buttons and visual feedback
  • Responsive design that works seamlessly across desktop and mobile devices
  • Empty state handling with encouraging graphics when no cards are present

Services & Tools Used:

  • Firebase Authentication – for secure user registration and login management
  • Firebase Firestore – for real-time card data storage with user-specific collections
  • CSS3 Animations – for advanced hover effects, transitions, and keyframe animations
  • CSS Custom Properties – for dynamic theming and consistent color management
  • HTML5 Canvas & SVG – for custom icons and scalable graphics
  • Modern Browser APIs – including Clipboard API for copy functionality
